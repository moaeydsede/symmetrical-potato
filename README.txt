مهم جداً (رفع من الموبايل بدون مجلدات)
================================
هذا المشروع (Flat) لا يحتوي على مجلدات — فقط ملفات في الجذر، حتى تقدر ترفعه من الموبايل بسهولة.

الملفات:
- index.html (المشاهد)
- editor.html (صفحة تحرير تساعدك توليد company.json)
- company.json (البيانات: الأرقام/المواقع/السوشيال)
- styles.css / app.js / qr.js / editor.js
- manifest.webmanifest / sw.js / icon.svg (PWA)

طريقة الرفع:
1) أنشئ Repo جديد على GitHub
2) Upload files → اختر كل الملفات (بدون مجلدات)
3) Settings → Pages → Deploy from branch → main / root
4) رابط المشاهد: https://USERNAME.github.io/REPO/
5) رابط التحرير: https://USERNAME.github.io/REPO/editor.html

التعديل الحقيقي (الذي ينعكس تلقائياً):
- افتح company.json داخل GitHub → Edit → Commit
ثم الموقع يتحدث تلقائياً ✅
