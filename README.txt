# Company Profile Website (GitHub Pages) - مجاني 100%

هذا الموقع يعمل على GitHub Pages بشكل مجاني بالكامل.

## 1) أين تعدّل البيانات؟
افتح الملف:
- `data.json`

وقم بتعديل:
- اسم الشركة `companyName`
- الشعار `logo`
- أرقام المبيعات `salesNumbers` (يمكن أكثر من رقم)
- رقم المصنع `factoryNumber` (اختياري)
- روابط السوشيال `social` (أي رابط فاضي لن يظهر)
- المواقع `locations` (أي عنصر بدون title+map لن يظهر)

## 2) تغيير اللوجو
ضع صورة اللوجو داخل:
- `images/logo.png`

أو غيّر مسار `logo` داخل `data.json`.

## 3) نشر الموقع على GitHub Pages
1. أنشئ Repository جديد (Public) باسم مثل: `company-site`
2. ارفع كل الملفات داخل هذا الـ repo
3. من GitHub:
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main` و Folder: `/ (root)`
4. بعد دقيقة ستجد رابط الموقع.

## 4) رابط المشاهدة vs رابط التعديل
- رابط المشاهدة: GitHub Pages (يعرض الموقع فقط)
- رابط التعديل: صفحة الـ repo على GitHub (لأصحاب الصلاحية فقط)

ملاحظة: للتعديل من الموبايل بسهولة يمكنك استخدام زر **Edit** داخل GitHub لكل ملف (خاصة data.json).


## 2) رابط المشاهدة vs رابط التعديل
- **رابط المشاهدة (للعملاء):** هو رابط GitHub Pages بعد النشر.
  مثال: `https://USERNAME.github.io/company-site/`

- **رابط التعديل (خاص لمن لديه صلاحية):** هو رابط المستودع على GitHub.
  مثال: `https://github.com/USERNAME/company-site`

> أي شخص معه رابط المشاهدة يقدر يشوف فقط.
> **التعديل لا يمكن إلا لمن لديه صلاحية على المستودع** (Owner/Collaborator).

## 3) التعديل بعد النشر بدون تعديل الأكواد
بعد نشر الموقع، للتعديل:
1) افتح المستودع على GitHub
2) افتح ملف `data.json`
3) اضغط زر القلم ✏️ (Edit)
4) عدّل الأرقام/الروابط/المواقع
5) اضغط Commit changes
خلال ثواني سيُحدّث الموقع تلقائياً.

## 4) إظهار رابط التعديل داخل الموقع (اختياري)
افتراضياً رابط التعديل **مخفي** حتى لا يظهر للمشاهدين.
إذا أردت إظهاره:
- في `data.json` ضع:
  - `showEditLink`: true
  - `editRepoUrl`: ضع رابط المستودع
